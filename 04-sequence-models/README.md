## 目录结构

```
04-sequence-models/
├── 01-rnn-basics/       # RNN 基础：LSTM、GRU、WaveNet、时间序列
```

| 子模块 | 核心概念 | 实践重点 |
|--------|----------|----------|
| RNN 基础 | 循环结构、梯度问题、时序预测 | 时间序列预测 |
