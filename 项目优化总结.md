## 🚀 使用方法

### 对于学习者

1. **开始学习**
   ```bash
   # 克隆项目
   git clone <repo-url>
   cd AI-Practices

   # 配置环境
   conda env create -f environment.yml
   conda activate ai-practices

   # 启动Jupyter
   jupyter notebook
   ```

2. **选择学习路径**
   - 阅读主README选择合适的路径
   - 参考模块README了解详细内容
   - 按照推荐顺序学习

3. **实践练习**
   - 运行所有notebook
   - 完成练习题
   - 尝试实际项目

### 对于贡献者

1. **了解规范**
   - 阅读 CONTRIBUTING.md
   - 学习 CODE_STYLE.md
   - 参考 NOTEBOOK_OPTIMIZATION_GUIDE.md

2. **优化现有内容**
   - 使用优化检查清单
   - 遵循代码规范
   - 补充文档说明

3. **提交贡献**
   - 创建分支
   - 提交改进
   - 发起Pull Request
