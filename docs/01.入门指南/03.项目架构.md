# 项目架构

AI-Practices 项目的整体架构和目录结构说明。

## 目录结构

```
AI-Practices/
├── 01-foundations/          # 机器学习基础
├── 02-neural-networks/      # 神经网络
├── 03-computer-vision/      # 计算机视觉
├── 04-sequence-models/      # 序列模型
├── 05-advanced-topics/      # 高级专题
├── 06-generative-models/    # 生成模型
├── 07-reinforcement-learning/ # 强化学习
├── 08-theory-notes/         # 理论笔记
├── 09-practical-projects/   # 实战项目
├── docs/                    # 文档网站
├── requirements.txt         # Python 依赖
└── README.md               # 项目说明
```

## 模块说明

### 基础模块

| 模块 | 内容 | 难度 |
|:-----|:-----|:----:|
| 01-foundations | 机器学习经典算法 | ⭐⭐ |
| 02-neural-networks | 深度学习基础 | ⭐⭐⭐ |

### 核心模块

| 模块 | 内容 | 难度 |
|:-----|:-----|:----:|
| 03-computer-vision | CNN、目标检测、分割 | ⭐⭐⭐ |
| 04-sequence-models | RNN、LSTM、Transformer | ⭐⭐⭐⭐ |

### 进阶模块

| 模块 | 内容 | 难度 |
|:-----|:-----|:----:|
| 05-advanced-topics | 优化、正则化、迁移学习 | ⭐⭐⭐⭐ |
| 06-generative-models | VAE、GAN、Diffusion | ⭐⭐⭐⭐⭐ |
| 07-reinforcement-learning | Q-Learning、Policy Gradient | ⭐⭐⭐⭐⭐ |

## 技术栈

```
┌─────────────────────────────────────────┐
│              应用层                      │
│  Jupyter Notebooks │ 实验脚本 │ 项目    │
├─────────────────────────────────────────┤
│              框架层                      │
│  PyTorch │ TensorFlow │ Keras │ JAX    │
├─────────────────────────────────────────┤
│              工具层                      │
│  NumPy │ Pandas │ Scikit-learn │ OpenCV│

│  Python │ CUDA │ cuDNN                  │
└─────────────────────────────────────────┘
```

## 学习建议

1. **循序渐进**：按模块顺序学习
2. **动手实践**：运行每个 Notebook
3. **理解原理**：先理论后代码
4. **项目驱动**：完成实战项目
