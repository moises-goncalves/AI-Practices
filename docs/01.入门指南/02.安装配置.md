# 安装配置

详细的环境配置和依赖安装指南。

## 环境要求

### 硬件要求

| 组件 | 最低配置 | 推荐配置 | 说明 |
|:-----|:--------|:---------|:-----|
| **CPU** | 4 核心 | 8+ 核心 | 数据预处理需要多核 |
| **RAM** | 8 GB | 32 GB | 大数据集需要更多内存 |
| **GPU** | GTX 1060 6GB | RTX 3080+ | 深度学习训练必需 |
| **存储** | 50 GB | 200 GB SSD | 数据集和模型存储 |

### 软件要求

| 软件 | 版本 | 说明 |
|:-----|:-----|:-----|
| **Python** | 3.10 - 3.11 | 推荐 3.10 |
| **CUDA** | 11.8+ | GPU 加速必需 |
| **cuDNN** | 8.6+ | 深度学习加速 |

## 安装方式

### 方式一：Conda 安装（推荐）

```bash
# 创建环境
conda create -n ai-practices python=3.10 -y
conda activate ai-practices

# 安装 PyTorch (CUDA 11.8)
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia

# 安装 TensorFlow
pip install tensorflow[and-cuda]

# 安装其他依赖
pip install -r requirements.txt
```

### 方式二：Docker 安装

```bash
# 拉取镜像
docker pull zimingttkx/ai-practices:latest

# 运行容器
docker run -it --gpus all -p 8888:8888 zimingttkx/ai-practices:latest
```

## 验证安装

```python
# 验证 PyTorch
import torch
print(f"PyTorch: {torch.__version__}")
print(f"CUDA Available: {torch.cuda.is_available()}")
print(f"GPU: {torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'N/A'}")

# 验证 TensorFlow
import tensorflow as tf
print(f"TensorFlow: {tf.__version__}")
print(f"GPU Devices: {tf.config.list_physical_devices('GPU')}")
```

## 常见问题

### CUDA 版本不匹配

```bash
# 检查 CUDA 版本
nvcc --version
nvidia-smi

# 重新安装对应版本的 PyTorch
pip uninstall torch torchvision torchaudio
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### 内存不足

```python
# 减少批量大小
batch_size = 16  # 从 64 减少到 16

# 启用混合精度训练
from torch.cuda.amp import autocast, GradScaler
scaler = GradScaler()
```
