快速开始

5 分钟内搭建开发环境并运行第一个示例

环境准备

系统要求

  • Python: 3.10+ (推荐 3.10.x)
  • 操作系统: Windows / macOS / Linux
  • 内存: 8GB+ (16GB 推荐)
  • GPU: NVIDIA GPU (可选，用于加速训练)

安装步骤

方式一：Conda 环境（推荐） 
# 1. 克隆仓库
git clone https://github.com/zimingttkx/AI-Practices.git
cd AI-Practices

# 2. 创建 Conda 环境
conda create -n ai-practices python=3.10 -y
conda activate ai-practices

# 3. 安装依赖
pip install -r requirements.txt

# 4. (可选) GPU 支持
pip install tensorflow[and-cuda]
# 或 PyTorch GPU 版本
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
方式二：pip 虚拟环境 
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

验证安装

Python 
# 检查 TensorFlow
python -c "import tensorflow as tf; print(f'TensorFlow: {tf.__version__}')"

# 检查 PyTorch
python -c "import torch; print(f'PyTorch: {torch.__version__}')"

# 检查 GPU
python -c "import tensorflow as tf; print(f'GPU: {tf.config.list_physical_devices(\"GPU\")}')"

运行第一个示例

Terminal 
# 启动 Jupyter Lab
jupyter lab

# 打开 01-foundations/01-training-models/ 目录
# 选择任意 .ipynb 文件开始学习

推荐学习顺序

  1. 从 01-foundations 开始，建立机器学习基础
  2. 进入 02-neural-networks 学习深度学习
  3. 根据兴趣选择 CV (03) 或 NLP (04) 方向
  4. 最后通过 09-practical-projects 实战检验
