# 01 - 机器学习基础

建立坚实的机器学习理论基础，掌握经典算法的原理与实现。

## 模块信息

| 属性 | 值 |
|:-----|:---|
| **前置要求** | Python, NumPy, 线性代数, 微积分 |
| **学习时长** | 4-6 周 |
| **Notebooks** | 25+ |
| **难度** | ⭐⭐ 初级到中级 |

## 学习目标

- ✅ 理解机器学习的核心概念和数学基础
- ✅ 实现线性回归、逻辑回归、SVM 等经典算法
- ✅ 掌握模型评估、交叉验证和超参数调优
- ✅ 理解集成学习并应用 XGBoost、LightGBM
- ✅ 应用降维和聚类等无监督学习技术

## 关键公式

**线性回归:**
```
ŷ = Xθ
θ = (XᵀX)⁻¹Xᵀy
```

**逻辑回归:**
```
σ(z) = 1 / (1 + e⁻ᶻ)
```

**SVM:**
```
minimize: ½||w||² + C Σξᵢ
```
