## Learning Paths

### Beginner Path

```
01-foundations → 02-neural-networks → 03-computer-vision
                                    ↘
                                      04-sequence-models
```

### Advanced Path

```
05-advanced → 06-generative
            ↘
              07-reinforcement-learning
```

### Practice Path

```
09-projects (includes Kaggle gold solutions)
```
