常见问题解答

学习过程中常见的问题和解决方案

环境配置问题

Q: 如何安装项目所需的全部依赖？

推荐使用 Conda 创建隔离环境：

Terminal
conda env create -f environment.yml
conda activate ai-practices

Q: 安装 TensorFlow 时报错怎么办？

  • 确保 Python 版本为 3.10.x
  • 升级 pip: pip install --upgrade pip
  • Mac M1/M2 芯片使用: pip install tensorflow-macos

Jupyter Notebook 问题

Q: Notebook 中无法导入已安装的包？

这通常是环境问题，为 Jupyter 注册环境：

Terminal
python -m ipykernel install --user --name=ai-practices

然后在 Notebook 中选择正确的 Kernel。

Q: Notebook 运行很慢或卡死？

  • 清理输出: Cell → All Output → Clear
  • 重启 Kernel: Kernel → Restart
  • 减小数据量进行测试

GPU 和 CUDA 问题

Q: 如何检查 GPU 是否可用？

Python
# TensorFlow
import tensorflow as tf
print("GPU 可用:", tf.config.list_physical_devices('GPU'))

# PyTorch
import torch
print("CUDA 可用:", torch.cuda.is_available())

Q: 训练时 GPU 内存不足 (OOM)?

  • 减小 batch_size
  • 使用混合精度训练
  • 设置 GPU 内存增长
Python
gpus = tf.config.experimental.list_physical_devices('GPU')
for gpu in gpus:
    tf.config.experimental.set_memory_growth(gpu, True)

模型训练问题

Q: 损失值为 NaN 或 Inf？

  • 降低学习率
  • 检查数据中是否有 NaN/Inf 值
  • 添加梯度裁剪: optimizer = Adam(clipnorm=1.0)

Q: 过拟合怎么办？

  • 添加正则化 (L1/L2)
  • 使用 Dropout
  • 数据增强
  • 早停 (Early Stopping)

学习路径问题

Q: 应该先学机器学习还是深度学习？

推荐先学机器学习基础，原因：

  • 理解基本概念（损失函数、优化、过拟合等）
  • 掌握数据预处理和特征工程
  • 深度学习是机器学习的子集

Q: 应该学 TensorFlow 还是 PyTorch？

  • 初学者：从 Keras 开始
  • 研究者：PyTorch 更灵活
  • 工程师：TensorFlow 部署方便
