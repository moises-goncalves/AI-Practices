# 01-Foundations | 机器学习基础

> 机器学习核心算法与理论基础

---

## 目录结构

```
01-foundations/
├── 01-training-models/          # 模型训练：梯度下降、正则化
├── 02-classification/           # 分类算法：MNIST 实战
├── 03-support-vector-machines/  # 支持向量机：线性/非线性/回归
├── 04-decision-trees/           # 决策树：训练、可视化、回归
├── 05-ensemble-learning/        # 集成学习：Bagging、Boosting、XGBoost
├── 06-dimensionality-reduction/ # 降维：PCA、Kernel PCA、LLE
├── 07-unsupervised-learning/    # 无监督学习：K-Means、DBSCAN
└── 08-end-to-end-project/       # 端到端项目：波士顿房价预测
```

---

## 学习路线

```
训练模型 → 分类 → SVM → 决策树 → 集成学习 → 降维 → 无监督 → 完整项目
```

---

## 核心内容

| 子模块 | 核心概念 | 实践重点 |
|--------|----------|----------|
| 训练模型 | 梯度下降、正则化、交叉验证 | 线性/多项式回归 |
| 分类 | 精度、召回率、ROC-AUC | MNIST 手写数字 |
| SVM | 核技巧、软间隔、SMO | 文本分类、图像 |
| 决策树 | 信息增益、CART、剪枝 | 可视化、回归树 |
| 集成学习 | Bagging、Boosting、Stacking | XGBoost 调参 |
| 降维 | PCA、t-SNE、流形学习 | 数据可视化 |
| 无监督 | 聚类、异常检测 | 客户分群 |
